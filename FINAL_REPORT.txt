QUICK START GUIDE

1. Navigate to Project:
   cd seaweather

2. Install Dependencies:
   pip install -r requirements.txt

3. Get API Key:
   Sign up at https://weatherstack.com/ (free tier)

4. Configure API Key:
   Edit .env file:
   WEATHERSTACK_API_KEY=your_key_here

5. Run Dashboard:
   python -m shiny run app.py

6. Access Dashboard:
   Open browser: http://localhost:8000

7. Use Dashboard:
   - Select country and city
   - Click "Fetch Weather"
   - View all weather, air quality, astronomy data
   - Check alerts and seasonal information

SUPPORT & TROUBLESHOOTING

Issue: "API key not set"
Fix:   Verify .env file contains WEATHERSTACK_API_KEY value

Issue: "No module named 'requests'"
Fix:   Run pip install -r requirements.txt

Issue: "Pandas not found"
Fix:   Run pip install pandas

Issue: "Connection timeout"
Fix:   Check internet connection or try again (API might be slow)

Issue: "Quota exceeded"
Fix:   Wait until next month or upgrade to paid plan
